  2. 第二个判断判断$page变量是不是在$whitelist变量中，在就返回true

  3. 第三个判断$page中?前面的字符是不是在$whitelist,在就返回true

  4. 第四个判断先把$page进行url解码，解码后判断$page中?前面的字符是不是在$whitelist，是就返回true

    而且include函数会以./or../过滤掉来包含文件
